Falcon(Beta)

C++, GLSL
Win32 API, SDL2, OpenGL 4.4 Core

It is a tool that captures a region of the screen and analyzes each pixel in parallel. The result can be projected on a window. After this, execute any action based on the results. It is useful to check the color like in FNF game. You can also activate a command from the keyboard or mouse.





Rabbit(Beta)

C++, HTML, CSS, JavaScript
wxWidgets, IE Engine

It is a tool to analyze different types of securities. You can adjust the number of spaces, element values, the sections of the items, the parameters to analyze, the number of canvases, etc. Uses the Internet Explorer engine. for simple DOM loading. It is useful for games that allow you trade things. It also has an HTTP request functionality for automatic updates for item values.
